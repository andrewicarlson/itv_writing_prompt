/********** Writing Prompt **********/

This is just a simple WordPress plugin that add's a writing prompt or helper to the post editing page.

To install, just upload the folder to your /wp-content/plugins/ directory on your WordPress site and in the admin panel, activate the plugin.

Version 1 includes:

-- Basic parts of an essay (thesis, primary and supporting arguments, conclusion and sources).
-- Tooltips or 'helpers' for each one to explain a little more about what each part of an essay does.

Version 1.1 plans:

-- Option to add additional fields for sources and arguments/statements.
-- Increased field data validation.
